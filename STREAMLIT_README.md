# 低功耗蓝牙信号优化系统 - Streamlit Web界面

## 🌟 功能特性

### 📊 主控面板
- **运行模式选择**: 支持优化、演示、监控、测试四种模式
- **参数配置**: 可调节信号采集时长、优化循环次数等参数  
- **一键运行**: 通过按钮交互替代命令行参数

### 📈 实时监控
- **性能指标**: 实时显示延迟、质量评分、内存和CPU使用率
- **趋势图表**: 动态更新的性能趋势图
- **进度跟踪**: 优化进度条和状态显示

### 📊 数据分析  
- **HDF5文件分析**: 自动识别和分析优化结果文件
- **信号可视化**: 增强信号波形、质量评分趋势等
- **特征矩阵**: 特征分布热图和统计信息

### 🧪 测试验证
- **信号生成**: 支持QPSK、OFDM、FSK等多种信号类型
- **参数调节**: 可配置信噪比、信号长度等参数
- **结果展示**: 时频域分析图表

## 🚀 快速开始

### 1. 启动应用

**方法一：使用脚本启动**
```bash
chmod +x run_streamlit.sh
./run_streamlit.sh
```

**方法二：直接启动**
```bash
# 激活虚拟环境
source .venv/bin/activate

# 启动应用
streamlit run streamlit_app_simple.py --server.port 8501
```

### 2. 访问界面

在浏览器中打开：http://localhost:8501

### 3. 使用指南

#### 🎯 优化模式
1. 在侧边栏设置参数（采集时长、循环次数等）
2. 点击"开始优化"按钮
3. 观察实时进度和质量评分
4. 查看优化完成后的趋势图表

#### 🎭 演示模式  
1. 点击"演示模式"按钮
2. 观看系统演示流程
3. 查看生成的示例图表

#### 📊 监控模式
1. 设置监控参数
2. 点击"监控模式"按钮  
3. 实时查看系统性能指标

#### 🧪 测试模式
1. 在"测试验证"标签页配置测试参数
2. 选择信号类型和参数
3. 点击"运行测试"查看结果

## 🛠️ 配置说明

### ⚙️ 系统参数
- **duration**: 信号采集时长 (0.01-1.0秒)
- **cycles**: 优化循环次数 (1-100次)
- **output_file**: 结果保存文件名

### 🎛️ 测试参数  
- **signal_type**: 信号类型 (qpsk/ofdm/fsk/noise)
- **snr_db**: 信噪比 (-10dB到30dB)
- **length**: 信号长度 (500-5000采样点)
